### Pretty QR Code Generator for [SedekahJe.com](https://sedekahje.com)

#### Generate QR Code

1. Install dependencies
   `bun install`

2. Run Nextjs app
   `bun dev`

3. Generate QR code
   `bun script.ts`

#### Important Files

**1. `/app/qr/[id]/page.tsx`**
This file is the styling of the QR code page.

**2. `/script.ts`**
Script that generate png photo.
